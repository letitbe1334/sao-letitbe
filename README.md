# sao-letitbe

> my kickass SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao letitbe my-project
```

### From git

```bash
sao letitbe1334/sao-letitbe my-project
```
